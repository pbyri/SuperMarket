********************************************************************
INSTRUCTIONS TO COMPILE / EXECUTE THIS APPLICATION
*******************************************************************

** Create a folder called "build" inside the root directory (mkdir build)
** cb build
** make (this will compile all targets), OR
** make SuperMarket (this will build only the executable SuperMarket target)
** make TestSuperMarket (this will build the unit test suite for SuperMarket)
** from the build directory, you can run the following commands
** ./src/SuperMarket (this will launch the application)
** ./tests/TestSuperMarket (this will run all the unit test suites on the application)
** ctest  (this will just execute the test suite and report single success or failure)
